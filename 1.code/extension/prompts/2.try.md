```typescript

const token = await context.secrets.get('teamsync.token');
```

```typescript
  // 인증

  POST /auth/refresh

  GET /settings/pull/latest
  GET /settings/pull/{id}
  GET /settings/history
  
  // 팀 관리

  POST /teams/{id}/invite
}
```
